# Shinchan Funny Name Generator

## English

This is a simple Python program that generates funny names by randomly combining two lists of first names and last names. Every time you run it, you get a different silly name inspired by characters from the Shinchan cartoon. This project is based on Chapter 1 of the book Impractical Python Projects by Lee Vaughan, published by No Starch Press. The original project used names from a TV show called Psych. I changed all the names to Shinchan characters to make my own version. From this project I learned how to use import to load Python tools, how to create a tuple, how to use random.choice() to pick random items, how to use a while True loop, how to stop a loop with break, how to combine text with .format(), and how to write clean code following PEP 8 rules. To run it, make sure Python 3 is installed, then type python3 shinchan_names.py in your terminal. Press Enter to get a new name and press n then Enter to quit.
